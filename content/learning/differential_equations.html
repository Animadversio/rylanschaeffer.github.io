Linear, First-Order Differential Equations

No Source Term

Consider the following linear, first-order differential equation where $y'(t)$ is shorthand for $\frac{dy(t)}{dt}$: $$y'(t) = a y(t) $$ The solution to this differential equations is one of the most important functions, the exponential function. To see that the exponential function is the correct solution to the above differential equation, observe that: $$y(t) = y(0) e^{a t} \Rightarrow y'(t) = a \, y(0) e^{a t} = a y(t) $$ Intuitively, what this means is that a system with these dynamics will either grow exponentially (for $a > 0$) or decay exponentially (for $a < 0$). For $a=0$, the system will sit unchanged.

Source Term

Now, suppose a source term $q(t)$ is added: $$y'(t) = a y(t) + q(t)$$ Every equation like this will have two solutions: the null solution and the particular solution. By null solution, I mean the solution $y_n(t)$ to the equation: $$ y_n'(t) - a y_n(t) = 0$$ By particular solution, I mean the solution $y_p(t)$ to the equation: $$ y_p'(t) - a y_p(t) = q(t)$$ The complete solution $y_c = y_p + y_n$ to this ODE is the sum of the null solution and the particular solution: $$ \begin{align*} 0 &= y_n'(t) - a y_n(t)\\ q(t) &= y_p'(t) - a y_p(t)\\ 0 + q(t) &= y_n'(t) - a y_n(t) + y_p'(t) - a y_p(t)\\ q(t) &= (y_n'(t) + y_p'(t)) - a(y_n(t) + y_p(t))\\ &= y_c'(t) - a y_c(t) \end{align*} $$ We should think of the null solution as the behavior of the system with no external source and the particular solution as the behavior of the system for the exact choice of external source $q(t)$.

A general solution to the equation $y'(t) - a y(t) = q(t)$ is the following: $$y(t) = y(0)e^{at} + \int_0^t e^{a(t-s)} q(s) ds $$ Before deriving this, let's try to gain some intuition for it. We see that if the source term is zero i.e. $q(t) = 0$, we recover the solution to the sourceless differential equation i.e. $y(t) = y(0)e^{at}$. As we expect, the starting condition $y(0)$ grows (decays) exponentially depending on whether $a > 0$ ($a < 0$). The second term is like a continuous sum of the new inputs, but each input $q(s)$ also grows/decays exponentially depending on how much time has elapsed since its occurence. For instance, if we're at $t=4$ and there was an input at time $t=2$, $q(s=2)$ , we would expect that input to have grown/decayed proportional to the elapsed two seconds i.e. $q(s=2) e^{a(4-2)}$. Delta Function

The delta function can be represented in different ways that are useful in specific contexts. Some of these forms are:
  • Gaussian: $\delta(x) = \lim_{\epsilon \rightarrow 0} \frac{1}{\sqrt{2 \pi \epsilon}} e^{-x^2 / 2 \epsilon^2}$

Solving Linear, First-Order ODEs with Integrating Factors

Are linear, first order ODEs typically solvable? The answer turns out to be yes, and we can show this using a tool called an integrating factor. To build up some intuition, consider solving the following ODE: $$y'(x) - a(x) y (x) = 0 $$ We can solve this ODE by putting the $y$ terms on one side and integrating: $$ \begin{align} \frac{y'(x)}{y(x)} &= a(x)\\ \log y(x) \lvert_{x_0}^x &= \int_{x_0}^x a(s) ds\\ y(x) &= y(x_0) + \exp(\int_{x_0}^x a(s) ds) \end{align} $$ If $a(x)$ is a constant, we see that we recover our previous solution, $y(x) = y(x_0) + \exp(a (x - x_0))$. The key insight was to see that we could isolate the $y$ terms and then integrate them independently from other variables. However, when we consider the general linear first-order ODE, there is no readily apparent method to isolate the $y$ terms. $$ a(x) y'(x) + b(x) y(x) = h(x)$$

If we could somehow group all the $y$ terms and turn it into the derivative of a function, then we could straightforwardly integrate. The idea is to multiply the ODE by a new function $u(x)$, called an integrating factor, that will allow us to rewrite the LHS as $(u(x) y(x))'$. If we can accomplish this, then we can integrate both sides, divide by u(x) and we will have our $y(x)$, as we did in the simpler case. To accomplish this, note that by the chain rule, $$ (u(x) y(x))' = u'(x) y(x) + u(x) y'(x)$$ and thus $$ u(x) y'(x) = (u(x) y(x))' - u'(x) y(x) $$ Plugging $u'(x) y(x)$ into our ODE, $$ a(x) (u(x) y(x))' - a(x) u'(x) y(x) + b(x) u(x) y(x) = u(x) h(x)$$ Dividing by $a(x)$: $$ (u(x) y(x))' - u'(x) y(x) + \frac{b(x)}{a(x)} u(x) y(x) = u(x) \frac{h(x)}{a(x)}$$ We need the second and third LHS terms to annihilate in order for the LHS to become $(u(x) y(x))'$. This will happen when: $$ u'(x) y(x) + \frac{b(x)}{a(x)} u(x) y(x) $$ This yields a much simpler ODE for $u(x)$ with solution: $$ u(x) = u(x_0) \exp (\int_{x_0}^x \frac{b(s)}{a(s)} ds)$$ Plugging back into our ODE: $$ \begin{align} (u(x) y(x))' - u'(x) y(x) + \frac{b(x)}{a(x)} u(x) y(x) &= u(x) \frac{h(x)}{a(x)}\\ (u(x) y(x))' &= u(x) \frac{h(x)}{a(x)}\\ (u(x) y(x)) &= \int_{x_0}^x u(s) \frac{h(s)}{a(s)} ds\\ y(x) &= \frac{1}{u(x)} \int_{x_0}^x u(s) \frac{h(s)}{a(s)} ds \end{align} $$

$u(x_0)$ doesn't matter since the final form of $y(x)$ is multiplied by $u(s)$ and divided by $u(x)$, so such a coefficient would cancel. This shows that any linear first-order ODE can be solved using such an integrating factor: $$ u(x) = \exp (\int_{x_0}^x \frac{b(s)}{a(s)} ds)$$

Linear Stability Analysis

For a continuous time dynamical system, we might want to understand whether, and how quickly, a point near a stable fixed point flows to the fixed point. Concretely, let $x$ be the phase space of some system and let $x^*$ be a fixed point of the system. We can define a small perturbation away from the fixed point as $\eta(t) \defeq x(t) - x^*$. Note that this is equivalent to defining a time-varying variable $x(t) = x^* + \eta(t)$. We'd like to understand whether this small pertubation, $\eta(t)$ grows or decays over time, and to answer such a question, we can construct a differential equation to describe the behavior of $\eta(t)$.

$\begin{align} \frac{d}{dt}\eta(t) &= \frac{d}{dt} (x(t) - x^*)\\ &= \frac{d}{dt} x(t)\\ \end{align}$

Using the definition of $x(t) = x^* + \eta(t)$, we can construct a Taylor Series expansion of $\frac{d}{dt} x(t)$ at $x^*$:

$\begin{align} \frac{d}{dt} x(t) &= \frac{d}{dt} (x^* + \eta (t))\\ &\approx \frac{d}{dt} x^* + \eta(t) \frac{d^2}{dt^2} x^* + O(\eta^2)\\ \end{align}$

Since $x^*$ is a fixed point, $\frac{d}{dt} x^* = 0$ and thus:

$\begin{align} \frac{d}{dt}\eta(t) = \eta(t) \frac{d^2}{dt^2} x^* \end{align}$

This is a linear equation in $\eta$ and is therefore called the "linearization about $x^*$". If $\frac{d^2}{dt^2} x^*$ is also zero, the $O(\eta^2)$ term that we previously ignored will become relevant. This linearization tells us that if the second derivative at $x^*$ is negative, then the perturbation will shrink, and inversely, if the second derivative is positive, the perturbation will grow larger. The absolute value of the second derivative, $|d^2 x^* / dt^2|$ tells us how quickly the pertubation will grow/shrink and its reciprocal $\frac{1}{|d^2 x^* / dt^2|}$ is called the characteristic time scale which expresses how much time is required to modify $x$ by a fixed amount around $x^*$.
