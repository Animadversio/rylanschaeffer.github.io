Calculus of Variations

Introduction

In calculus, we're familiar with the notion of a function's derivative: $$ \frac{df(x)}{dx} = \lim_{\delta \rightarrow 0} \frac{f(x+\delta) - f(x)}{\delta} $$ In calculus of variations, we are interested in a related concept called a functional's derivative. A functional $F[f(x)]$ is a function that accepts a function $f(x)$ as its input and returns a scalar value. One example of a functional is the entropy $H[]$ of a probability density function p(x). Just how we typically use the derivative to identify which points $x$ extremize the function $f(x)$, calculus of variations concerns identifying which functions $f(x)$ extremize the functional $F[f(x)]$. Continuing with the previous example, we may want to know what probability distribution maximizes or minimizes entropy. How might we define a functional derivative?

The goal of a functional derivative is to express how the scalar output of the functional changes as the input function $f(x)$ changes at each point $x$ in the considered domain $[a, b]$. Consequently, we start with modifying $f(x)$ by adding some arbitrary function $\eta(x)$ scaled by an infinitesimally small quantity $\delta$, and ask what the functional $F[f(x) + \delta \eta(x)]$ outputs. To a first order approximation at a point $x$, the functional changes by some amount $\frac{dF[f(x)]}{f(x)}$ at that point, and since we moved the function $f(x)$ by $\delta \eta(x)$ at that point, we expect the functional to change by $\delta \eta(x) \frac{dF[f(x)]}{f(x)}$. To quantify the total change, we integrate (sum) over all points. We formalize this using the functional's Taylor series expansion: $$F[f(x) + \delta \eta(x)] = F[f(x)] + \delta \int dx \, \eta(x) \frac{dF[f(x)]}{f(x)} + O(\delta^2)$$ Rearranging and taking the limit of $\delta \rightarrow 0$, we see that: $$\lim_{\delta \rightarrow 0} \frac{F[f(x) + \delta \eta(x)] - F[f(x)]}{\delta} = \int dx \, \eta(x) \frac{dF[f(x)]}{f(x)}$$ and we call $\frac{dF[f(x)]}{f(x)}$ the functional derivative. Now, consider trying to find a function $f(x)$ that minimizes the functional $F[f(x)]$. Like with function derivatives, we require that at a minimum, the limit equals 0. $$ 0 = \lim_{\delta \rightarrow 0} \frac{F[f(x) + \delta \eta(x)] - F[f(x)]}{\delta} = \int dx \, \eta(x) \frac{dF[f(x)]}{f(x)}$$ The only way for this to hold true for arbitrary function $\eta(x)$ is for the functional derivative to vanish at all points i.e. $\forall x, \frac{dF[f(x)]}{f(x)} = 0$.
