Continuous Functions

Definitions of Continuity

Epsilon-Delta Definition of Continuity

Let $f(x)$ be a scalar function. We say that $f$ is continuous at the point $x_0$ if $$\forall \epsilon > 0, \exists \, \delta_{\epsilon} > 0 \text{ s.t. } |x - x_0| < \delta \Rightarrow |f(x) - f(x_0)| < \epsilon$$

Definitions of Uniform Continuity

Barbalat's Lemma

Consider a scalar, differentiable function $f(t)$. We might ask whether the following properties hold:

  • Does $\lim_{t \to \infty} \dot{f}(t) = 0 \Rightarrow f(t)$ converges? No. Consider $f(t) = \sin(\log(t)) \Rightarrow \dot{f}(t) = \frac{1}{t} \cos(\log(t))$. The time derivative converges to 0, but the function itself oscillates forever as a sinusoid.
  • Does $f(t)$ converges $\Rightarrow \lim_{t \to \infty} \dot{f}(t) = 0$? No. Consider $f(t) = e^{-t} \cos(e^{t}) = \Rightarrow \dot{f}(t) = \cos(e^t)$. The function converges to 0, but the time derivative oscillates forever.

Given both these negative answers, Barbalat's Lemma aims to answer the question, under what conditions can we state that the derivative of a function vanishes? Before revealing the answer, we need to review a property called uniform continuity.
